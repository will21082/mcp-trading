CHIẾN LƯỢC MẶC ĐỊNH: BREAKOUT STRATEGY

Cách hoạt động:
┌────────────────────────────────────────────────────────┐
│ 1. Tìm BB Squeeze (BBW < 0.03)                        │
│    → Bollinger Bands đang thu hẹp                     │
│                                                        │
│ 2. Chờ Breakout                                        │
│    → Giá vượt upper BB + change >2%                   │
│    → HOẶC giá dưới lower BB + RSI <30                 │
│                                                        │
│ 3. Volume Confirmation                                 │
│    → Volume >100k                                      │
│                                                        │
│ 4. RSI Healthy (30-70)                                │
│    → Không quá mua/bán                                │
│                                                        │
│ 5. Trend Strong (ADX >25)                             │
│    → Trend đủ mạnh                                    │
│                                                        │
│ 6. BB Rating +1 to +3                                 │
│    → Tín hiệu mua từ BB                               │
│                                                        │
│ → Tổng điểm ≥5: TẠO SIGNAL                           │
└────────────────────────────────────────────────────────┘

Entry Example:
  Entry:  $0.001300
  SL:     $0.001261 (3% risk)
  TP1:    $0.001358 (+4.46%) - Chốt 50%
  TP2:    $0.001417 (+9.00%) - Chốt 30%
  TP3:    $0.001475 (+13.46%) - Chạy 20%
  R:R:    1:3

💡 VÍ DỤ 3 CHIẾN LƯỢC BẠN CÓ THỂ TẠO:
═══════════════════════════════════════════════════════════════════

1. RSI Mean Reversion
   Entry: RSI <25 + Volume >2x + Price >EMA50
   Exit: RSI reaches 50-60
   R:R: 1:2

2. EMA Crossover
   Entry: EMA50 >EMA200 + Price >EMA50 + ADX >25
   Exit: Price crosses below EMA50
   R:R: 1:3

3. Volume Breakout
   Entry: Volume >5x avg + Change >3% + RSI <70
   Exit: Fixed 9% TP
   R:R: 1:3

→ Xem chi tiết trong CREATE_YOUR_STRATEGY.md

Q: Làm sao thay đổi timeframe?

   Hoặc sửa trong strategy file

A: Có! Run từng strategy riêng và compare signals

Q: Làm sao biết signal nào tốt?
A: Xem confidence score
   ≥8: Rất tốt
   6-7: Tốt
